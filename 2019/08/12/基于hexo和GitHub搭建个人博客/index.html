基于hexo和GitHub搭建个人博客

发表于 2019-08-12 | 更新于: 2019-08-12 | 分类于 创建博客
字数统计: 627

前期准备

　

  1. 下载node.js
    　　下载 node.js
    　　选择左边的长期支持版（Recommend For Most Users）

  2. 下载git
    　　下载git 　
    　　根据电脑自行选择32位或64位，配置path路径啥的自行百度即可

  3. 使用管理员权限打开命令行
    　　接下来的代码都再命令行实现

  4. 利用npm安装淘宝镜像源cnpm

  5. 安装hexo

正式安装

  1. 新建一个文件夹（我这里新建的文件夹叫blog）

  2. 初始化博客

　　安装完成后blog文件中会自动生成一些文件

  1. 启动博客

　　会生成一个本地地址http://localhost:4000， 复制这个地址在浏览器中打开，会发现博客已经建好，并且自动生成了一个hello world博文

  1. 新建文章

　　可以在blog/source/_posts目录下面看到已经生成的博客

  1. 写博客
    　　打开文章，基于markdown格式编写（自行百度），这里可以先随便写点东西测试，具体格式可以后期在钻研
    　　保存以后先hexo清理

    再hexo生成

　　再重新启动hexo

　　启动完成后刷新一下刚刚的网页，刚刚写的博客就出现在网页中了

部署到GitHub

  1. 注册登录GitHub

  2. 点右上角的＋号新建一个仓库（New repository）,在Repository name处填写：你的昵称.github.io（这里的命名必须是这个格式，例如我的是：jsslrks.github.io，后期访问博客只要输入这个地址即可），然后create即可

  3. 在blog目录下安装一个git部署的插件

  4. 设置blog文件中的_config.yml文件
    　　到文件的最底部，找到#Deployment，在这个小模块的type:后面配置
    　　type: git
    　　repo: https://github.com/jsslrks/jsslrks.github.io （这边回到第2步的网页复制一下Repository name里的网址，粘贴过来）
    　　branch: master

  5. 部署到远端

　　输入你的用户名和密码，输入密码时不会显示东西，但是你是输入进去的，输入完放心回车即可

  1. 访问你的博客
    　　新建标签页，在浏览器网址栏输入：你的用户名.github.io（就是刚刚的Repository name），就成功部署到GitHub上了

布置博客

到这里博客就搭建完成了，博客的美观可以根据自己的喜好自行设置
